# Linux System Info Toolkit

## Description
A Bash script designed to automate the collection of essential system information, including User details, Hostname, IP Address, and Kernel version. This tool is optimized for RHCSA tasks, quick server audits, and troubleshooting.

## Usage
To run the script, ensure it is executable and run:

```bash
chmod +x sysinfo.sh
./sysinfo.sh
